# 📊 Deloitte Australia Data Visualization using Tableau

## Project Overview

This project was completed as part of the **Deloitte Australia Data Analytics Virtual Experience Program**. The objective was to analyze factory telemetry data using Tableau and create interactive visualizations to identify operational downtime trends and support data-driven decision-making.

## Business Problem

Manufacturing facilities experience equipment downtime that impacts operational efficiency. This project analyzes downtime across factories and device types to identify performance bottlenecks and support maintenance planning.

---

## Objectives

- Analyze downtime across factories.
- Identify device types causing the most downtime.
- Create interactive dashboards for operational insights.
- Present findings through effective data visualization.
